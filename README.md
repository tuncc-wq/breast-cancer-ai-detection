# Breast Cancer Detection using Machine Learning

## Project Overview
This project uses a Random Forest model to classify whether a tumor is malignant or benign using medical data.

## What I did
- Loaded real-world breast cancer dataset
- Trained a machine learning model
- Evaluated performance using:
  - Accuracy
  - Confusion Matrix
  - Precision / Recall / F1-score

## Results
The model achieved high accuracy and strong recall, meaning it can detect most cancer cases effectively.

## Tech Stack
- Python
- Scikit-learn
- Matplotlib

## Goal
This project demonstrates how AI can assist in early cancer detection.
